# SEO Content Information Architecture

Use this skill whenever the task affects page composition, internal linking, search landing pages, or on-page SEO structure.

## Core rules

- search-facing pages need clear purpose
- internal linking should support discoverability
- headings and copy structure should communicate topic intent cleanly
- low-value combinations should not create index bloat

## Required concerns

- heading structure
- internal link paths
- category and landing page intent
- facet page value versus noindex strategy
- content hierarchy and supporting copy

## Review checklist

Reject or fix:
- thin search-facing pages with no real topic value
- faceted pages generating low-value indexable combinations
- weak internal linking to important content
- unclear topic hierarchy
