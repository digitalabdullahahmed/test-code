# SEO Monitoring Search Operations

Use this skill whenever the task affects SEO QA, search diagnostics, or operational monitoring of crawl and metadata behavior.

## Core rules

- SEO is operational, not just implementation
- validate what search engines actually see
- monitor for drift in metadata, indexing, and discoverability

## Required concerns

- metadata QA
- robots and sitemap validation
- canonical correctness checks
- search-facing route audits
- indexing anomaly review

## Review checklist

Reject or fix:
- no process to validate search-facing output
- metadata or canonical drift left unchecked
- sitemap entries not matching index intent
- crawl-blocked important routes
