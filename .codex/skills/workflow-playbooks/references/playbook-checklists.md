# Playbook Checklists

## New feature

- identify surfaces
- choose skills
- inspect first
- implement
- verify
- summarize

## Bug fix

- isolate fault
- fix minimal scope
- add regression coverage
- verify

## Backend module

- define module boundary
- define contracts
- implement
- test
- verify

## Frontend feature

- define UX and ownership
- implement route and module cleanly
- handle loading and error states
- verify responsiveness and accessibility

## SEO

- classify index intent
- define metadata
- define canonical
- validate output
